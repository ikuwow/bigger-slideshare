## Supported Browsers

* Chrome

## Installation

### Chrome

* Download [latest release](https://github.com/ikuwow/larger-slides/releases/latest)
* Unarchive
* Open `chrome://extensions`
* `LOAD UNPACKED` and select downloaded directory
